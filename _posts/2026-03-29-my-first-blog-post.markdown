## 为什么选择 Jekyll？

Jekyll 是一个简单的静态站点生成器，非常适合用来搭建个人博客。它有以下优点：

- **简单易用**：使用 Markdown 编写文章
- **免费托管**：可以直接部署到 GitHub Pages
- **高度可定制**：支持主题和插件
- **快速加载**：生成的是纯静态 HTML 文件
